This weeks content was an introduction to Arduino! Ardunio is an open-source electronics platform based on easy-to-use hardware and software. WEEK 9

PERSONAL PROJECT IDEATION!

I've decided I'll go with a web-based project. Thematically, my idea is a comment on how stress and anxiety can control a person like how a person controls a computer.

WHAT

COMPUTER RPG GAME: navigate a social interaction! In a website - clicking changes the screens. Interact with person but the computer controls your interaction, disabling the "human" interaction.

WHY

Idea that human-computer interaction is controlled by the human interacting - so what if the computer is the one controlling your interactions?

HOW

Through visual studio code and p5. I will create a p5.js drawing app as the main interactive part of the project at the end. I want to use a lot of animated gifs in the website and utilise digital illustration + give the user the option to digitally illustrate themselves!
